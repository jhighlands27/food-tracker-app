ğŸ’¬ SMS NOTIFICATIONS (OPTIONAL SETUP)
--------------------------------------

Currently notifications print to console. To enable real SMS:

1. Sign up for Twilio (free trial available):
   https://www.twilio.com/

2. Install Twilio:
   pip install twilio

3. In main.py, find the `send_sms_notification` function

4. Uncomment the Twilio code and add your credentials:
   account_sid = "your_account_sid"
   auth_token = "your_auth_token"
   twilio_number = "+1234567890"

5. Restart server

Users with phone numbers will now get real SMS alerts!

ğŸ› TROUBLESHOOTING
------------------

Problem: Camera won't start
Solution: Check browser permissions, use HTTPS in production

Problem: Images won't upload
Solution: Make sure uploads/ folder exists

Problem: Stats show $0.00
Solution: Add prices when creating items

Problem: Database error after update
Solution: Delete food_tracker.db and restart (recreates with new schema)

Problem: Scanner doesn't detect barcode
Solution: Ensure good lighting, hold camera steady, try manual entry
